# Marconi Perillo 2026 - Apresentação Estratégica

Apresentação interativa de inteligência política para a campanha de Marconi Perillo ao Governo de Goiás em 2026.

## Tecnologias

- Next.js 13.5
- React 18
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Deploy

O projeto está configurado para deploy automático na Vercel.

## Navegação

- Use as **setas do teclado** (← →) para navegar entre slides
- **Barra de espaço** avança para o próximo slide
- Clique nos **indicadores** na barra inferior para ir direto a um slide
